Running It

Running JerrOS is quite simple. Ensure you have qemu installed (specifically the i386 version, because that is what JerrOS runs on):

Debian

$ sudo apt install qemu-system-i386

Arch

$ sudo pacman -S qemu

Fedora

$ sudo dnf install qemu-system-i386

macOS

$ brew install qemu

Windows

get a linux computer.

Once that is completed, we can make and run.

$ make
$ chmod +x run.sh
$ ./run.sh

And there you have it! JerrOS running in all its glory.
